# new

### USAGE

```
$ grill new [OPTIONS] [PATH]
```

### ARGS

`PATH`

Where to create the new package. Defaults to the current directory.

`--name <NAME>`

Name of the package.

### OPTIONS

`--lib`

Set project to be a static library.

`--gui`

Set project to be a GUI application.

`--bin` (default)

Set project to be a console application.
